#Stickyitem v.1.0.0

jQuery helper plugin, adds class to elements when window is scrolled to it's position.

**Requirements:**

* jQuery

**How to use:**

``` JavaScript
if ($.fn.stickyitemSetup) {
  $('.js-scrollto').stickyitemSetup({
    stickyClass: 'is-sticky'
	});
}
```
